# Read a file with APEX export of fångster from Spöreg database and do some cleanup

The function uses file extension to select how the file should be read.

## Usage

``` r
read_fangst_clean()
```

## Value

Return a tibble

## Examples

``` r
if (FALSE) { # \dontrun{
fangst <- read_fangst_clean()
} # }
```
